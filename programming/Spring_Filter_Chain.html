# Spring Filter Chain

스프링 필터는 체인구조로 되어있다.


서블릿 체인에서 스프링에 등록된 기본 filter들 (doFilter로 등록) 을 체크한다. 
여기에 새로운걸 추가하려면 (ex Spring Security) VirtualFilterChain 형태로 등록해주면 좋다 (Spring에 구현되어있음)
